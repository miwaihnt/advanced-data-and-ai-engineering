# Hybrid Search Evaluation Report

このドキュメントは、ベクトル検索（Semantic Search）と全文検索（BM25/FTS）の特性を比較・分析し、ハイブリッド検索の有用性を検証するためのものである。

## 1. 比較検証: クエリ「予算」

| 項目 | ベクトル検索 (Meaning) | 全文検索 (Keywords) |
| :--- | :--- | :--- |
| **主な結果** | 予算の具体的な内訳を説明する長文スピーチ | 「一般会計予算」等の議題を宣言する形式的な発言 |
| **強み** | 単語が直接含まれていなくても、「予算に関連するトピック」を広く拾える | クエリの単語そのものが目立つ文書をピンポイントで拾える |
| **弱み** | 似た内容の重複（例: 同一人物の別日の挨拶）が上位を独占しやすい | 中身が薄くても単語さえ入っていれば上位に来てしまう |

### 1.1. 実測結果の証拠 (Evidence)

#### Vector Search Result #1 (Distance: 0.2232)
> ○井上副大臣　令和五年度予算につきまして、ただいま財務大臣から説明いたしましたとおりでありますが、若干の点について補足説明いたします。... 社会保障関係費につきましては、年金給付費十三兆八百五十七億円、医療給付費十二兆一千五百十七億円 ... 合計三十六兆八千八百八十九億円を計上しております。

#### Full-Text Search Result #1 (BM25 Score: 3.4534)
> ○根本委員長　これより会議を開きます。令和五年度一般会計予算、令和五年度特別会計予算、令和五年度政府関係機関予算、以上三案を一括して議題とし、審査に入ります。

### 1.2. ベクトル検索の詳細分析
- **特徴**: 予算の詳細（社会保障費、防衛費などの金額）を羅列している補足説明が上位。
- **考察**: モデル（E5-small）は「予算」という単語の有無よりも、その周辺に出現する「計上」「歳出」「兆円」といった関連語の密度（文脈）を高く評価している。情報密度（Content-rich）を重視する傾向がある。

### 1.3. 全文検索（FTS）の詳細分析
- **特徴**: 会議の冒頭などで「予算を議題とする」と宣言している一文が上位。
- **考察**: BM25アルゴリズムにより、文書内での「予算」という単語の集中度が高く評価されている。会議の構成要素（Title/Topic）を特定するのに適している。

## 2. トークナイズ戦略の評価

SudachiPy による「事前分かち書き（Pre-tokenization）」の導入により、以下の効果を確認した。

- **正確なマッチング**: `令和 五 年度 一般 会計 予算` のように分解されることで、日本語特有の複合語も正確なキーワードとして認識されている。
- **ノイズの低減**: 助詞（「は」「の」など）をフィルタリングする余地が生まれ、検索の純度を高めることが可能となった。

## 3. ハイブリッド検索（RRF）への展望

今回の分析から、単一の検索手法では以下のトレードオフがあることが判明した。
- ベクトル検索：**「何について話しているか（Content）」** に強いが、多様性に欠ける。
- 全文検索：**「どのキーワードが出たか（Word）」** に強いが、中身の薄いものも拾う。

**RRF (Reciprocal Rank Fusion)** による統合により、両手法の「順位」を掛け合わせ、「形式的に正しく、かつ内容も伴っている」文書を上位に押し上げることが期待される。
